### 使用说明
ForcePress是IPFS原力区旗下转为用户方便的编辑，部署去中心化博客与网站的工具

#### 1- 下载
```
git clone https://github.com/eoshackathon/forcepress.git
```

#### 2- 安装
```
cd forcepress
npm install
```

#### 3- 运行
```
hexo serve -d
```

然后在浏览器的地址栏中输入:
```
http://localhost:4000/admin
```
打开浏览器，编辑博客

#### 4- 部署到IPFS网络
点击`Deploy`菜单，一键部署
